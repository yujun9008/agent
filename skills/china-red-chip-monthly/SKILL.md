# 中证红利成分股月度表现查询

## 功能

1. 使用官方成分股列表（/scripts/000922cons.xls，共100只）
2. 查询指定月份每个成分股的涨跌幅
3. 涨跌幅计算：月初开盘价 → 月末收盘价
4. 按跌幅排序，输出前10只股票

## 输入

用户需要提供：
- 年月，如 "2026年2月" 或 "2026-02"

## 执行方式

```bash
python3 ~/.agents/skills/china-red-chip-monthly/scripts/monthly_performance.py <年份-月份>
```

示例：
```bash
python3 monthly_performance.py 2026-02
```

## 数据来源

- 成分股列表：中证红利指数官方Excel文件（000922cons.xls）
- 股票行情：新浪财经API (`money.finance.sina.com.cn`)

## 注意事项

1. 涨跌幅计算采用前复权价
2. 上月末收盘价：指定月份上一个月的最后一个交易日收盘价
3. 月末收盘价：指定月份最后一个交易日收盘价
4. 成分股列表更新时需替换 000922cons.xls 文件
