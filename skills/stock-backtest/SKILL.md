# 股票组合回测

## 功能

1. 根据提供的股票代码列表和日期范围
2. 计算每只股票在该期间的涨跌幅（显示股票名称）
3. 计算等权配置组合的总收益率

## 输入

用户需要提供：
- 股票代码列表（逗号分隔），如 `600737,601963,002572`
- 开始日期，如 `2026-03-02`
- 结束日期，如 `2026-03-13`

## 执行方式

```bash
python3 ~/.agents/skills/stock-backtest/scripts/backtest.py <股票代码列表> <开始日期> <结束日期>
```

示例：
```bash
# 10只跌幅前10的股票，回测2026年3月2日到3月13日
python3 backtest.py "600737,601963,002572,601318,601288,601398,600755,002867,600729,000651" 2026-03-02 2026-03-13
```

## 输出示例

```

股票数量: 10 只（等权配置）

600737 中粮糖业: 2026-03-02 收盘 15.97 → 2026-03-13 收盘 15.74  -1.44%
601963 重庆银行: 2026-03-02 收盘 9.84 → 2026-03-13 收盘 11.28  +14.63%
...

============================================================
回测结果
============================================================
| 代码 | 名称 | 涨跌幅 |
|------|------|--------|
| 600737 | 中粮糖业 | -1.44% |
| 601963 | 重庆银行 | +14.63% |
...

**组合总收益率: +1.18%**
============================================================
```

## 计算方式

- 期初价格：开始日期当日的收盘价
- 期末价格：结束日期当日的收盘价
- 股票名称：通过新浪财经API实时获取
- 组合总收益率：各股票涨跌幅的均值（等权配置）
