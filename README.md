# Agent Skills

这个仓库包含自定义的 Agent Skills，用于股票数据分析。

## Skills

### 1. china-red-chip-monthly

查询中证红利指数（sh000922）成分股在某月份的表现，输出跌幅前10的股票。

**功能：**
- 使用官方成分股列表（100只股票）
- 查询指定月份每个成分股的涨跌幅
- 涨跌幅计算：上月最后一个交易日收盘价 → 月末收盘价
- 按跌幅排序，输出前10只股票

**使用方式：**
```bash
python3 scripts/monthly_performance.py <年份-月份>
# 示例
python3 scripts/monthly_performance.py 2026-02
```

**输出示例：**
```
## 2026年02月 中证红利成分股表现

| 排名 | 代码 | 名称 | 月涨跌幅 |
|------|------|------|----------|
| 1 | 600737 | 中粮糖业 | -11.39% |
| 2 | 601963 | 重庆银行 | -6.67% |
...

**共计 99 只成分股，平均涨跌幅: +1.84%**
```

### 2. stock-backtest

股票组合回测，计算一段时间内等权配置组合的总收益率。

**功能：**
- 根据提供的股票代码列表和日期范围
- 计算每只股票在该期间的涨跌幅
- 计算等权配置组合的总收益率

**使用方式：**
```bash
python3 scripts/backtest.py <股票代码列表> <开始日期> <结束日期>
# 示例
python3 scripts/backtest.py "600737,601963,002572" 2026-03-02 2026-03-13
```

**输出示例：**
```
回测期间: 2026-03-02 → 2026-03-13
股票数量: 3 只（等权配置）

600737: 2026-03-02 收盘 15.97 → 2026-03-13 收盘 15.74  -1.44%
601963: 2026-03-02 收盘 9.84 → 2026-03-13 收盘 11.28  +14.63%
002572: 2026-03-02 收盘 13.59 → 2026-03-13 收盘 13.13  -3.38%

**组合总收益率: +3.27%**
```
